# 🎲 Número da Sorte

![.NET MAUI](https://img.shields.io/badge/.NET-MAUI-512BD4?style=for-the-badge&logo=dotnet)
![C#](https://img.shields.io/badge/C%23-Programming-239120?style=for-the-badge&logo=csharp)
![Status](https://img.shields.io/badge/status-em%20estudo-blue?style=for-the-badge)

Aplicativo mobile multiplataforma desenvolvido com **.NET MAUI**, com o objetivo de explorar conceitos fundamentais de desenvolvimento com C# e interfaces modernas.

Este app gera um **"número da sorte" aleatório**, servindo como base para aprendizado de eventos, UI e organização de código.

---

## ✨ Funcionalidades

- 🎲 Geração de número aleatório
- 📱 Interface simples e intuitiva
- ⚡ Resposta instantânea ao clique
- 🔄 Atualização dinâmica da UI

---

## 🧰 Tecnologias e ferramentas

- .NET MAUI
- C#
- XAML
- Visual Studio 2022+

---

## 🧠 Conceitos abordados

Este projeto cobre fundamentos importantes:

- Manipulação de eventos (`Clicked`)
- Geração de números aleatórios (`Random`)
- Data Binding básico
- Estrutura de apps MAUI
- Separação entre UI e lógica

---

## 📂 Estrutura do projeto
NumeroDaSorte/
│
├── MainPage.xaml # UI principal
├── MainPage.xaml.cs # Lógica da aplicação
├── App.xaml # Configurações globais
├── AppShell.xaml # Navegação
├── Platforms/ # Código específico por plataforma
└── Resources/ # Imagens, fontes, estilos

## 🚀 Como executar

### Pré-requisitos

- Visual Studio 2022 ou superior
- Workload .NET MAUI instalado

### Passo a passo

# Clone o repositório
git clone https://github.com/rodrigomgrassioto/curso-dotnet-MAUI-Elias-Ribeiro-NumeroDaSorte.git
Abra o projeto no Visual Studio
Escolha uma plataforma (Android, Windows, etc.)
Execute (F5)
